# Install Gorgon

Add Gorgon to your project and reduce server load **fast**.

## NPM
```bash
npm install @mikevalstar/gorgon
```

## Yarn
```bash
yarn add @mikevalstar/gorgon
```

## PNpM
```bash
pnpm add @mikevalstar/gorgon
```
