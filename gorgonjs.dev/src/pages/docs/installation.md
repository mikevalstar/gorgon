# Install Gorgon

Add Gorgon to your project and reduce server load **fast**.

## NPM
```bash
npm install @gorgonjs/gorgon
```

## Yarn
```bash
yarn add @gorgonjs/gorgon
```

## PNpM
```bash
pnpm add @gorgonjs/gorgon
```
